Equality operators

"==" vs "==="

"==" is the equality operator. It checks if the values on either side of the operator are equal. It perform type coercion. i.e "18" == 18 is true.
"===" is the strict equality operator. It checks if the values on either side of the operator are equal and of the same type. It does not perform type coercion. i.e "18" === 18 is false.
